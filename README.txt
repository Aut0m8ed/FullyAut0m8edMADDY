# FullyAut0m8edMADDY - Phemex Bot Backend

## Setup Instructions

1. Rename `.env.example` to `.env`
2. Add your Phemex API credentials:
   PHEMEX_API_KEY=your_key_here
   PHEMEX_API_SECRET=your_secret_here

3. Run the server:
   npm install
   node index.js

4. Test endpoint:
   POST /api/test-trade
